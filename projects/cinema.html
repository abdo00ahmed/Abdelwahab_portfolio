Cinema — Movie Discovery

FlutterREST APIClean Architecture

</> Overview

Cinema is a movie discovery app built as a team project during the NTI Flutter Diploma. It consumes the TMDB REST API to surface a catalog of 500,000+ titles, with real-time search and advanced filtering, and was developed in an Agile workflow using Git, GitHub, and Jira.

✓ Core Capabilities

  • Real-time search and advanced filtering across the TMDB catalog
  • Caching strategy that cut network fetch time by 40%
  • Clean Architecture with strict separation of data, domain, and presentation layers
  • Team-based Git workflow with code review via Jira tickets
