Tasky — Task Manager

FlutterFirestore

</> Overview

Tasky is a task management app built around fast, real-time sync. Tasks written on one device appear near-instantly on another via Firebase Firestore, backed by secure session handling through Firebase Auth.

✓ Core Capabilities

  • Sub-second real-time task sync via Firestore
  • Secure session management with Firebase Auth
  • Redesigned task-creation flow, cutting the number of steps roughly in half
  • Full feature accessibility preserved despite the simplified flow
