Volt — AI Fitness Coach

FlutterGemini AIBloc

</> Overview

Volt is an AI-powered fitness app that generates personalized workout and meal plans on demand. It integrates Google's Gemini model via Firebase AI Logic, with a strict JSON schema enforced on every response so the app can render structured plan data reliably instead of parsing free-form text.

✓ Core Capabilities

  • Personalized workout and meal plan generation via the Gemini API
  • Structured JSON schema enforcement for consistent, parseable AI output
  • Dedicated screens for workout detail and meal plan breakdowns
  • Bloc/Cubit state management with a resilient parsing pipeline
  • Weekly progress tracking with calculated completion metrics
