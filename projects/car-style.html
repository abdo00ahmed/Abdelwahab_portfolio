Car Style — Workshop Manager

FlutterFirebaseNode.jsPaymob SDK

</> Overview

Car Style is a Flutter workshop management system built for a real freelance client. It delivers three complete role-based experiences in one app: customers book and track repairs, mechanics manage assigned jobs, and admins oversee the whole operation — order flow, technician assignment, and payments — from a dedicated dashboard.

✓ Core Capabilities

  • Three role-based portals — Admin, Mechanic, Customer — each with a tailored dashboard
  • Real-time order tracking and technician assignment via Firestore
  • Native Paymob SDK integration through an Android Method Channel for in-app payments
  • Custom Node.js/Express backend deployed on Railway, driving Firebase Cloud Messaging alerts
  • Push notifications for order updates and assignment changes across all three roles
  • Clean Architecture with the MVI pattern and Bloc/Cubit for predictable state
